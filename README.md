# rbxchat

rbxchat is a Discord bot that automatically fetches and provides the latest release of the rbxchat executable from GitHub.

The bot retrieves release information using the GitHub API and provides download, documentation, and issue links directly inside Discord.

---

## Features

- Automatically fetches the latest GitHub release
- Provides direct download links
- Shares documentation link
- Shares issues/support link
- Uses fallback URLs if GitHub API is unreachable
- Simple environment-based configuration

---

## Requirements (Developer)

- Node.js v18 or newer
- A Discord bot application
- GitHub repository with releases
